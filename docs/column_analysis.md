# Understand what each column tells us

## Sales Performance
1. Sales
2. Profit
3. Quantity
4. Discount

### These tell us:
- Are we selling enough?
- Are we making money?
- Are discounts hurting profits?

---

## Customer Analysis
1. Customer ID
2. Customer Name
3. Segment

### These tell us:
- Who are our best customers?
- Which customer segments generate most revenue?

---

## Product Analysis
1. Product ID
2. Product Name
3. Category
4. Sub-Category

### These tell us:
- Which products sell well?
- Which products lose money?

---

## Geographic Analysis
1. Country
2. Region
3. State
4. City

### These tell us:
- Which locations perform best?
- Which locations need attention?

---

## Operations Analysis
1. Order Date
2. Ship Date
3. Ship Mode

### These tell us:
- Are deliveries fast?
- Which shipping mode is used most?

---
